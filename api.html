Documentation pour les Revendeurs

Automatisez vos commandes en connectant votre site ou votre propre panel SMM directement à ViralFlows.

URL de Base (Endpoint)

POST https://script.google.com/macros/s/TON_ID_DE_SCRIPT/exec

Authentification

Tous les appels nécessitent votre clé API passée en paramètre key.

1. Ajouter une commande

Passez une commande pour un lien spécifique.

Paramètres requis :
action = api_add_order
key = VOTRE_CLE_API
service = ID_DU_SERVICE
link = https://instagram.com/p/xxx
quantity = 1000

Réponse JSON Succès

{
  "status": "success",
  "order_id": "API-1708123456",
  "message": "Commande reçue avec succès."
}

2. Voir son solde (Wallet)

action = api_balance
key = VOTRE_CLE_API 
{
  "status": "success",
  "balance": 145.50,
  "currency": "USD"
}
