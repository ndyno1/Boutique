Documentation API

Automatisez vos commandes en connectant votre site ou votre panel SMM directement à ViralFlows.

URL de Base (Endpoint)

POST https://script.google.com/macros/s/AKfycbx_Tm3cGZs4oYdKmPieUU2SCjegAvn-BTufubyqZTFU4geAiRXN53YYE9XVGdq_uq1H/exec

Authentification

Tous les appels nécessitent votre clé API passée en paramètre key.

1. Lister les services

Récupère la liste de tous les services disponibles avec vos tarifs revendeur appliqués.

Paramètres requis :
action = api_services
key = VOTRE_CLE_API

Réponse JSON Succès

{
  "status": "success",
  "services": [
    {
      "service": "12",
      "name": "Abonnés Instagram Garantis",
      "category": "Boost SMM",
      "rate": 1.50,
      "min": "100",
      "max": "10000"
    }
  ]
}

2. Ajouter une commande

Passez une commande pour un lien spécifique. Le montant sera déduit de votre Wallet.

Paramètres requis :
action = api_add_order
key = VOTRE_CLE_API
service = ID_DU_SERVICE
link = https://instagram.com/p/xxx
quantity = 1000

Réponse JSON Succès

{
  "status": "success",
  "order_id": "API-1708123456",
  "message": "Commande reçue avec succès."
}

3. Statut d'une commande

Vérifie l'état d'une commande passée via l'API.

Paramètres requis :
action = api_status
key = VOTRE_CLE_API
order_id = ID_DE_LA_COMMANDE

Réponse JSON Succès

{
  "status": "success",
  "order_id": "API-1708123456",
  "order_status": "En traitement",
  "remains": 0
}

4. Voir son solde (Wallet)

Récupère le solde actuel de votre portefeuille.

Paramètres requis :
action = api_balance
key = VOTRE_CLE_API

Réponse JSON Succès

{
  "status": "success",
  "balance": 145.50,
  "currency": "USD"
}
