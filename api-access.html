Documentation Rapide

Point d'entrée (Endpoint URL)

POST https://script.google.com/macros/s/AKfycbx_Tm3cGZs4oYdKmPieUU2SCjegAvn-BTufubyqZTFU4geAiRXN53YYE9XVGdq_uq1H/exec

1. Récupérer la liste des services

Récupère tous les services disponibles avec vos tarifs appliqués.

action = api_services key = VOTRE_CLE_API
Réponse: {"status":"success", "services":[{"service":"12", "name":"Followers IG", "rate":1.50, ...}]}

2. Passer une commande

Crée une nouvelle commande et débite automatiquement votre wallet.

action = api_add_order key = VOTRE_CLE_API service = ID_DU_SERVICE (ex: 12) link = URL_A_BOOSTER quantity = 1000

3. Vérifier son solde (Wallet)

Récupère le solde actuel de votre portefeuille en USD.

action = api_balance key = VOTRE_CLE_API
Réponse: {"status":"success", "balance":145.50, "currency":"USD"}
